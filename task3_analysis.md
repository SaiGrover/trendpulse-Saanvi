# 📊 Task 3: Exploratory Data Analysis (EDA)

This document presents the analysis performed on the cleaned dataset obtained after data preprocessing.

---

## 📌 Dataset Overview

The dataset contains real-time news articles fetched from an API and cleaned for analysis.

### Key Columns:

* **title** → Headline of the news article
* **source** → News publisher
* **author** → Author of the article
* **published_at** → Timestamp of publication
* **description** → Short summary
* **content** → Full/partial content
* **title_length** → Length of headline
* **content_length** → Length of content
* **date** → Publication date
* **hour** → Hour of publication

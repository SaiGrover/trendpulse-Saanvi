# TrendPulse – Real-Time Data Pipeline

TrendPulse is a Python-based data pipeline project that fetches real-time trending news data from an API, processes and cleans it, performs exploratory data analysis (EDA), and visualizes key insights.

## Features
- Live data extraction using API
- Data cleaning and preprocessing
- Feature engineering (time, content length, etc.)
- Exploratory Data Analysis (EDA)
- Visualizations of trends and patterns

## Tech Stack
- Python
- Pandas
- Matplotlib
- Requests

## 📂 Project Structure
- task1_data_collection.py → Fetches live data
- task2_data_processing.py → Cleans and processes data
- task3_analysis.py → Performs EDA
- task4_visualization.py → Generates visual insights

## Output
- Cleaned dataset
- Trend analysis (sources, timing, content patterns)
- Graphs for insights

## Objective
To build an end-to-end data pipeline that transforms raw real-time data into meaningful insights.
